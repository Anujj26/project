# Social Links Profile

A modern, responsive "Social Links Profile" page built with HTML, CSS, and a little JavaScript.

## Features

- Clean dark UI with glassmorphism card styling
- Responsive centered layout
- Profile image with circular framing
- Social media buttons with icons
- Smooth hover, glow, and press animations
- Optional dark/light theme toggle
- Single-page frontend with separated files

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts (Inter)

## Project Structure

```text
.
|-- index.html
|-- style.css
|-- script.js
|-- profile.jpg.png
`-- README.md
```

## Preview

Open `index.html` in your browser to run the project locally.

## Customization

- Update social links in `index.html`
- Edit styles in `style.css`
- Adjust theme toggle behavior in `script.js`
